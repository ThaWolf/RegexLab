# <span className="text-3xl font-bold text-garden-dark dark:text-earth-light">Regular Expression Documentation</span>

## <span className="text-xl font-semibold text-garden-dark dark:text-earth-light">What is a regular expression?</span>

A regular expression is a pattern that allows you to search and manipulate text flexibly. They are used to validate data or find matches within strings.

## <span className="text-lg font-semibold text-garden-dark dark:text-earth-light">Basic symbols</span>

- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">.</code> any character
- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">*</code> zero or more repetitions
- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">+</code> one or more repetitions
- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">?</code> optional
- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">[abc]</code> character set
- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">\d</code> digit
- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">\w</code> alphanumeric character
- <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">( )</code> group

## <span className="text-lg font-semibold text-garden-dark dark:text-earth-light">Examples</span>

- Validate a simple email: <code className="bg-sky-light dark:bg-sky-dark px-2 py-1 rounded">/^[\w.-]+@[\w.-]+\.[A-Za-z]{2,}$/</code>
- Extract numbers: <code className="bg-sky-light dark:bg-sky-dark px-2 py-1 rounded">/\d+/g</code>
- Check date format <code className="bg-earth-light dark:bg-garden-dark px-2 py-1 rounded">dd/mm/yyyy</code>: <code className="bg-sky-light dark:bg-sky-dark px-2 py-1 rounded">/^(0[1-9]|[12][0-9]|3[01])\/(0[1-9]|1[0-2])\/\d{4}$/</code>
